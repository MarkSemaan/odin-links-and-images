# Links and Images

## Links

To create a link in HTML, we use the anchor element. An anchor element is defined by wrapping the text or another HTML element we want to be a link with an `<a>` tag.

### Href and Target

Href is the value of the anchor tag telling where the link will go to, and target controls behavior of opening that link.

### Absolute and Relative Links

Absolute links point to other pages on the internet, relative links point to another page relative to the page you are clicking the link on.
